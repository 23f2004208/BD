# BD

## Birthday Greeting Card

An interactive birthday greeting card with an envelope animation featuring a warm brown color palette.

## Color Palette

This project uses a cohesive brown-themed color scheme:

- **Primary Dark (brown-900)**: `#4E342E`
- **Primary (brown-700)**: `#6D4C41` - Used for headings and primary accents
- **Primary Light (brown-500)**: `#8D6E63` - Used for highlights and closing text
- **Light (brown-300)**: `#BCAAA4`
- **Very Light (brown-100)**: `#EDE0D0` - Letter background
- **Background/Surface**: `#EFEBE9` - Envelope body
- **Text Primary**: `#3E2723` - Main text color
- **Text Secondary**: `#5D4037`
- **Border/Muted**: `#D7CCC8` - Envelope flap
- **Accent/Highlight**: `#A1887F`

The gradient background uses `#8D6E63` to `#6D4C41` for a warm, elegant appearance.
